[Aporte] CleverCss en sublime text

2012/11/22

Navegando por internet en busqueda de alguna manera de realizar “ZenCoding” pero en documentos css, me tope con CleverCSS una libreria en python para escribir css al “estilo” python y luego compilarlo. Esto me hizo acordar mucho a Sass y Less, con la ventaja que no se necesita un compilador extra, asi que decidi crear una version muy alpha, para poder utilizarlo en el Sublime Text.Por lo tanto, cualquier consejo, arreglo o sugerencia sera bienvenida.

(Para utilizarlo seleccionan el texto, abren la paleta de comandos (ctrl+shift+p) y elijen “Format with CleverCSS” )
